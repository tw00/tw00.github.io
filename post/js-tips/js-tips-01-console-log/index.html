JS Tip 01 - Minimalistic Logging

04 Nov 2019

Real pro’s debug with console.log ;-)

So, here is a quick tip, how you can add logging to a JS script, with ZERO dependencies. Just define log as:

const log = (...args) => console.log('[context]', ...args)

To disable logging, change it to:

const log = () => ()

or just uncomment it:

const log = (...args) => () // console.log('[context]', ...args)

also you can have log levels like this:

const log1 = (...args) => console.log('[context]', ...args)
const log2 = (...args) => console.warn(...args)
const log3 = (...args) => console.error(...args)

and easily disable / enable logging by:

const _no_log = () => ()
const _log = (...args) => console.log(...args)

const log1 = _log;
const log2 = _no_log;
const log3 = _no_log;
